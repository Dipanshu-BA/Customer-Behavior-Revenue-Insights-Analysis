# 📊 Customer Behavior & Revenue Insights Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior to uncover **actionable insights** related to revenue trends, customer segmentation, and product performance.

It follows an **end-to-end analytics workflow**:
- Data Cleaning & Transformation (Python)
- Data Analysis (SQL)
- Data Visualization (Power BI)

The objective is to simulate a real-world business scenario and enable **data-driven decision-making**.

---

## 🎯 Objective
- Understand customer purchasing behavior  
- Identify high-value customer segments  
- Analyze product and category performance  
- Evaluate the impact of discounts and subscriptions  
- Build an interactive dashboard for insights  

---

## 🛠️ Tech Stack
- **Python** (pandas, numpy) → Data cleaning & feature engineering  
- **SQL (MySQL)** → Data analysis (CTEs, window functions)  
- **Power BI** → Dashboard & visualization  

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning & Feature Engineering (Python)
- Handled missing values using **category-based median imputation**  
- Standardized column names  
- Created features:
  - Age Groups (segmentation)  
  - Purchase Frequency (in days)  
- Removed redundant columns  
- Loaded cleaned data into MySQL  

---

### 2️⃣ Data Analysis (SQL)
- Solved **10+ business queries**, including:
  - Revenue by gender and age group  
  - Top-performing products and categories  
  - Discount effectiveness  
  - Subscriber vs non-subscriber behavior  
  - Customer segmentation (New, Returning, Loyal)  
- Used:
  - **CTEs (Common Table Expressions)**  
  - **Window Functions (ROW_NUMBER)**  

---

### 3️⃣ Data Visualization (Power BI)
- Built an **interactive dashboard** with:
  - KPIs: Total Customers, Avg Purchase, Avg Rating  
  - Filters: Category, Gender, Subscription, Shipping Type  
- Visualized:
  - Revenue by category  
  - Customer distribution by age group  
  - Subscription trends  

---

## 📊 Key Insights
- 👕 Clothing category generates the highest revenue and sales  
- 📉 Only **27% customers are subscribed**, showing growth opportunity  
- 👥 Young Adults contribute the most revenue  
- 💸 Discounts still result in higher purchase amounts  
- ⚠️ Outerwear category underperforms  

---

## 💡 Business Impact
- Helps identify **high-value customer segments**  
- Supports **pricing and discount strategies**  
- Improves **customer retention planning**  
- Enables **data-driven decision-making**  

---

## 🚀 How to Run
1. Run the Python notebook for data cleaning  
2. Load cleaned data into MySQL  
3. Execute SQL queries  
4. Open Power BI dashboard
